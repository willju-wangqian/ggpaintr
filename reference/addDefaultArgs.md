# Add one element into a list of arguments

Add one element into a list of arguments

## Usage

``` r
addDefaultArgs(defaultArgs, ui_element, ui_param, plot_settings)
```

## Arguments

- defaultArgs:

  the default argument list

- ui_element:

  the keyword of the ui element

- ui_param:

  the parameter of the ui function

- plot_settings:

  a list contains all ui elements

## Value

a list of default arguments
