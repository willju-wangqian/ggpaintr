# Build the Tabbed Control UI for a Parsed Formula

Build the Tabbed Control UI for a Parsed Formula

## Usage

``` r
paintr_get_tab_ui(paintr_obj, copy_rules = NULL)
```

## Arguments

- paintr_obj:

  A `paintr_obj`.

- copy_rules:

  Effective or user-supplied copy rules.

## Value

A Shiny UI object or `NULL`.
