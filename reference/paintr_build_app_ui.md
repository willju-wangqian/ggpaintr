# Build the Standard ggpaintr App UI

Build the Standard ggpaintr App UI

## Usage

``` r
paintr_build_app_ui(title_label, draw_label, export_label)
```

## Arguments

- title_label:

  App title text.

- draw_label:

  Draw button text.

- export_label:

  Export button text.

## Value

A Shiny UI object.
