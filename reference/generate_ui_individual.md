# Build UI for One Placeholder

Build UI for One Placeholder

## Usage

``` r
generate_ui_individual(key, id, param, layer_name = NULL, copy_rules = NULL)
```

## Arguments

- key:

  Placeholder keyword.

- id:

  Placeholder id.

- param:

  Parameter label.

- layer_name:

  Layer name.

- copy_rules:

  Effective or user-supplied copy rules.

## Value

A Shiny UI object or `NULL`.
