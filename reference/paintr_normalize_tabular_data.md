# Coerce an Object to Tabular Data for `ggpaintr`

Coerce an Object to Tabular Data for `ggpaintr`

## Usage

``` r
paintr_normalize_tabular_data(data, source = "Data")
```

## Arguments

- data:

  An arbitrary R object.

- source:

  A short label for error messages.

## Value

A `data.frame`-like object with normalized column names.
