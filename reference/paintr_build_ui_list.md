# Build a UI List for a Parsed Formula

Build a UI List for a Parsed Formula

## Usage

``` r
paintr_build_ui_list(paintr_obj, copy_rules = NULL)
```

## Arguments

- paintr_obj:

  A `paintr_obj`.

- copy_rules:

  Effective or user-supplied copy rules.

## Value

A named list of UI controls by layer.
