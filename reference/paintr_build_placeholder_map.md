# Build Placeholder Metadata for a Parsed Formula

Build Placeholder Metadata for a Parsed Formula

## Usage

``` r
paintr_build_placeholder_map(
  keywords_list,
  id_list,
  param_list,
  index_path_list,
  placeholders
)
```

## Arguments

- keywords_list:

  A parsed keyword list by layer.

- id_list:

  A parsed id list by layer.

- param_list:

  A parsed parameter list by layer.

- index_path_list:

  A parsed index-path list by layer.

- placeholders:

  An effective placeholder registry.

## Value

A named list of placeholder metadata records by layer.
