# Add Layer Checkboxes to a UI List

Add Layer Checkboxes to a UI List

## Usage

``` r
ui_insert_checkbox(ui, nn, copy_rules = NULL)
```

## Arguments

- ui:

  A list of UI elements for one layer.

- nn:

  The layer name.

- copy_rules:

  Effective or user-supplied copy rules.

## Value

A list of UI elements, possibly prefixed with a checkbox.
