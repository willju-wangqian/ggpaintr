# Build Text Placeholder UI

Build Text Placeholder UI

## Usage

``` r
generate_ui_text(id, param, layer_name = NULL, copy_rules = NULL)
```

## Arguments

- id:

  Placeholder id.

- param:

  Parameter label.

- layer_name:

  Layer name.

- copy_rules:

  Effective or user-supplied copy rules.

## Value

A `textInput()` UI object.
