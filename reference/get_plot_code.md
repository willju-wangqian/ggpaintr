# Extract code and plot

Extract code and plot

## Usage

``` r
get_plot_code(componentList, data, data_path, data_filter = "")
```

## Arguments

- componentList:

  list of plot components and code components

- data:

  data

- data_path:

  path to the data

- data_filter:

  data filtering code

## Value

a named list of two elements; plot and code
