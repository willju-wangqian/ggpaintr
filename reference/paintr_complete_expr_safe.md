# Safely Complete a Parsed Formula

Safely Complete a Parsed Formula

## Usage

``` r
paintr_complete_expr_safe(paintr_obj, input, envir = parent.frame())
```

## Arguments

- paintr_obj:

  A `paintr_obj`.

- input:

  A Shiny input-like object.

- envir:

  The evaluation environment.

## Value

A structured runtime result.
