# Build Numeric Placeholder UI

Build Numeric Placeholder UI

## Usage

``` r
generate_ui_num(id, param, layer_name = NULL, copy_rules = NULL)
```

## Arguments

- id:

  Placeholder id.

- param:

  Parameter label.

- layer_name:

  Layer name.

- copy_rules:

  Effective or user-supplied copy rules.

## Value

A [`numericInput()`](https://rdrr.io/pkg/shiny/man/numericInput.html) UI
object.
