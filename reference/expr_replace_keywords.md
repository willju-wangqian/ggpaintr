# Replace One Placeholder in an Expression

Replace One Placeholder in an Expression

## Usage

``` r
expr_replace_keywords(.expr, keyword, index_path, input_item)
```

## Arguments

- .expr:

  An expression object.

- keyword:

  A placeholder symbol.

- index_path:

  An index path.

- input_item:

  A user input value.

## Value

The updated expression.
