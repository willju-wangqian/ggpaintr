# Build the Standalone App Template

Build the Standalone App Template

## Usage

``` r
get_shiny_template(formula_text, copy_rules = NULL, placeholders = NULL)
```

## Arguments

- formula_text:

  A single formula string.

- copy_rules:

  Effective or user-supplied copy rules.

- placeholders:

  Optional custom placeholder definitions or an effective placeholder
  registry.

## Value

A character vector containing the exported app template.
