# Build Dynamic `var` Controls for a Parsed Formula

Build Dynamic `var` Controls for a Parsed Formula

## Usage

``` r
output_embed_var(input, output, paintr_obj, envir = parent.frame())
```

## Arguments

- input:

  A Shiny input object.

- output:

  A Shiny output object.

- paintr_obj:

  A `paintr_obj`.

- envir:

  Environment used to resolve local data objects.

## Value

A named list of generated `var` UI controls.
