# ggpaintr: Formula-Driven ggplot Shiny Apps

`ggpaintr` provides the `ggpaintr` workflow for turning a single
ggplot-like formula string into a small Shiny app with generated
controls, rendered plots, generated code, upload support, and inline
runtime feedback.
